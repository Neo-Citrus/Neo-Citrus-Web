Welcome to my first blog post! 😄 Even though you're seeing this post over my website, I've actually written it over at Obsidian! 

I have automatised blog posts so I can just write them there and then they'll be automatically uploaded to Neo Citrus Web. It's not that hard to do, actually!

- Make a new **Obsidian Vault.**
- Set it so it's **inside of your website documentation** and name it whatever you want! Since I'm doing blog posts, mine's called "Blog".
- *Voilá*, you're all good to go! 😄

(Ideally you should also set whatever coding app you're using to ignore the *.obsidian* file through JavaScript, since you don't really need it to show up).

I hope that helps out to whoever is working on a website, or at least that you found it interesting!
